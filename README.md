# Orion VCS

Orion is a Git-inspired snapshot-based version control system built in Node.js.  
It implements core Git internals including blobs, trees, commits, branching, and checkout.

## Features
- Content-addressable object storage (SHA-1)
- Blob, tree, and commit objects
- Branching and history traversal
- Working tree status and checkout
- Colored CLI interface
- Cross-platform support

## Install
```bash
npm install -g orion-vcs
